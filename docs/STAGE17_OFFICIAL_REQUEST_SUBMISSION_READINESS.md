# Stage 17 遠賀川河川事務所データ照会の送信準備

## 現在地

Stage 17取得経路Aは承認済みである．公開公式DBの取得・監査は継続できる．一方，遠賀川河川事務所への実送信には，研究者本人を特定する情報，研究利用条件，対象期間，送信方法の確定が必要である．

公式の「事務所アクセス・お問い合わせ」ページで，電話 `0949-22-1830`，所在地 `〒822-0013 福岡県直方市溝堀1丁目1-1`，一般連絡メール `onga@qsr.mlit.go.jp` を確認した．データ照会専用窓口とは明記されていないが，公式ドメイン上の直接連絡経路である．現時点の推奨は，完全な本文と依頼者情報を固定したうえでこの公式メールへ送り，必要に応じて担当部署への転送または別手続を依頼する方法である．

## 教授から必要な情報

### 1．依頼者情報

- 所属大学
- 学部・研究科・部局
- 研究室名
- 氏名
- 職位
- 連絡先メールアドレス
- 連絡先電話番号
- 必要に応じて郵送先住所

### 2．研究情報

- 正式な研究課題名
- 資料の利用予定期間
- 成果公開範囲
  - 査読論文
  - 学会発表
  - 卒業論文・修士論文・博士論文
  - 研究室Web
  - softwareまたは公開dataset
- 研究費・共同研究等の課題番号がある場合はその名称と番号

### 3．必要な資料の対象期間

次のいずれかを選ぶ必要がある．

- 指定期間だけを依頼する
- 利用可能な全期間を依頼し，そのうち優先期間を指定する

優先期間には開始日と終了日が必要である．河床測量については測量実施年を限定せず，最新資料と過去資料の両方の利用可否を照会できる．

### 4．送信方法

- 電話で担当部署を確認し，指定された公式メールまたは資料提出経路へ送る
- 確認済みの公式メールへ送る
- 確認済みの公式問い合わせフォームから送る
- 郵送する

現時点の推奨は，公式メール `onga@qsr.mlit.go.jp` への文書送付である．電話確認を先に行う場合，本環境には電話発信機能がないため，教授または大学関係者が行う必要がある．

現行の自動送信準備gateが許可候補として扱うのは，公式ページ原本とSHA-256を封印した上記メール経路だけである．電話，フォーム，郵送へ変更する場合は，後続送信先を含む方式別の公式証拠を追加し，gateを別途審査するまでreadyにしない．

## 照会する資料

1．河床測量・鉛直基準面

- 定期横断測量，河床横断，河床縦断，距離標，断面位置座標
- 測量年月日，精度，鉛直基準面
- 河口堰上下流，魚道，西川・曲川合流部を含む電子資料

2．水位・流量観測

- 観測所一覧，座標，水位・流量時系列
- rating curveまたは水位流量換算関係
- 観測間隔，欠測flag，品質flag，鉛直基準面
- 曲川の既存観測の有無

3．河口堰

- 8門それぞれの有効幅，敷高，開度・操作時系列
- 上下流水位，操作時刻・理由
- 流量係数または既往水理資料

4．魚道

- 平面・縦断・横断形状，敷高，制御部形状
- 運用状態，流量，水位差―流量関係
- 既往流速測定・水理検討資料

5．独立Validation資料

- 河口堰，魚道，両合流部，河口側のADCPまたは流速計観測
- 観測時刻，測線，水深，東西・南北流速，品質管理情報

## fail-closed規則

送信準備validatorは，依頼者情報，正式課題名，利用予定期間，成果公開範囲，資料優先期間，送信方法，本文最終承認，送信先公式確認のいずれかが欠けていれば，`readyForSubmission=false`とする．本文承認はbooleanだけでなく，送信する文面そのもののSHA-256，承認者，承認日時へ結合する．送信先確認も，選択方法，実際の宛先，公式確認元URLとSHA-256，確認者，確認日時へ結合する．メール，フォーム，郵送，電話確認の選択と実宛先が一致しなければ停止する．このgateは物理値のsolver採用，境界割当，physical run，公開シミュレータ接続を許可しない．
